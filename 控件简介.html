控件简介

控件简介

布局

布局主要是用来控制控件的摆放位置关系

QT 中布局主要有两种，QVBoxLayout 与 QHBoxLayout

其中 V = Vertical (垂直的)，所有控件按照加入 Layout 的顺序依次在垂直方向上摆放

其中 H = Horizontal (水平的)，所有控件按照加入 Layout 的顺序依次在水平方向上摆放

详细用法参见：QVBoxLayout ,QHBoxLayout

按钮

QT 中充当按钮的控件为 QPushButton

激活按钮

使用按钮去触发函数使用 QT 的信号与槽机制

例如：使用 "Save" 按钮去触发 Save() 函数

private slots:
    void Save()
    {
        //pass;
    }

QPushButton *save = new QPushButton("SAVE"); 

connect( save , SIGNAL(sigClicked()) , this , SLOT(Save()));

详细用法参见：QPushButton

下拉菜单

QT 中充当按钮的控件为 QComboBox

常用方法

// 获取当前选中的文字
QComboBox->currentText() 

// 添加选项
QComboBox->addItem()

详细用法参见：QComboBox

分组框

QT 中充当按钮的控件为 QGroupBox

常用方法

// 添加控件
QGroupBox->addWidget();
// 添加布局
QGroupBox->addLayout();

详细用法参见：QGroupBox

标签

QT 中充当标签的控件为 QLabel

常用方法

// 初始化一个标签
QLabel *label = new QLabel("标签");

详细用法参见：QGroupBox

单行文本框

QT 中充当单行文本框的控件为 QLineEdit

常用方法

// 获取文本框中的内容
QLineEdit->text()

详细用法参见：QLineEdit

表格

QT 中充当表格的控件为 QTableWidget
